## App preview

Visit [https://testflix.surge.sh](https://testflix.surge.sh)

## Project setup
```
git clone https://github.com/adeviss2/testflix.git testflix
cd testflix
yarn install
```

### Compiles and hot-reloads for development
```
yarn serve
```

### Compiles and minifies for production
```
yarn build
```
